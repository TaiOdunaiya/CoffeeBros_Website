# Coffee Bros Website

A modern, responsive website for Coffee Bros, a coffee shop that showcases their menu, customer reviews, and contact information.

## Features

- 🎨 Modern and responsive design
- ☕ Interactive menu section
- 📱 Mobile-friendly layout
- 📸 Image slider using Swiper.js
- 📞 Contact form for table bookings
- ⭐ Customer testimonials section
- 📍 Location and business hours information
- 🛒 Shopping cart integration (UI only)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [Swiper.js](https://swiperjs.com/) - For image carousel
- [Font Awesome](https://fontawesome.com/) - For icons
- Google Fonts - For typography

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/CoffeeBros-Website.git
```

2. Navigate to the project directory:
```bash
cd CoffeeBros-Website
```

3. Open `index.html` in your web browser to view the website.
